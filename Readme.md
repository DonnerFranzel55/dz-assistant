# DZ Assistent
The DZ Assistent is a virtual assistent written in HTML, CSS and JS.

## Features
The assistent can:
- tell jokes
- tell the weather*
- tell the time and date

* API Key required

## Demo
You can test the assistent [here](https://donnerfranzel55.github.io/dz-assistant/).
